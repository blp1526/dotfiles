# Dotfiles

## Usage

### Host OS (OSX)

* as non-root user
  * Run `./provisioner/shared/as_user.sh`

### Guest OS (Fedora)

If you use VirtualBox, read `cheatsheets/virtualbox.md`.

* as root user
  * Run `./provisioner/fedora/as_root.sh`
  * Run `./provisioner/fedora/network.sh` (optional)

* as non-root user
  * Run `./provisioner/shared/as_user.sh`
