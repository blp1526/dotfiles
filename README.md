# Dotfiles

## Usage

Ubuntu

```
$ sudo ./provisioners/root_ubuntu.sh
$ ./provisioners/user_common.sh
```

macOS

```
$ ./provisioners/user_macos.sh
$ ./provisioners/user_common.sh
```
